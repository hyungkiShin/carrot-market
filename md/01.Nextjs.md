### NextJS
- 풀스택 앱을 만들기 위한 모든게 들어있는 프레임워크. 
- API 를 만들기 위해 다른 서버를 구축할 필요가 없다.
- NextJS 만으로 우리가 전에 했던 모든 프론트 엔드 기능을 만드는 동시에 API 까지 만들 수 있다는점.
- Pages 하위에 api 폴더를 만드는 순간 NextJS 서버에 API 가 생긴다.
